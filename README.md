# Lichess Stats Backend

## 📌 Description

Lichess Stats Backend is a Java-based backend application that retrieves chess data from the Lichess public API, stores it in a database, and provides statistical insights for players.

The system allows users to analyze game history, win rates, and most common openings.

## 🎯 Project Goal

The purpose of this project is to practice backend development using Java and gain experience with:

* REST API design
* Working with external APIs
* Database integration
* Data processing and aggregation

## ⚙️ Tech Stack (planned)

* Java
* Spring Boot
* REST API
* PostgreSQL (or H2 for development)
* Maven

## 📊 Features (MVP)

* Fetch player data from Lichess API
* Store games in database
* Provide player statistics:

  * total games
  * wins / losses / draws
  * most played openings

## 🧱 Domain Model (planned)

* Player
* Game
* Opening

## 🚀 Future Improvements

* Filtering by rating and time control
* Advanced statistics
* Opening analysis
* Integration with frontend
